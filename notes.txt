# GraphQL Tutorial from https://www.youtube.com/watch?v=bX2e4FILf78

### Stack

Front End
- React
- Apollo

Backend
- Express.js
- GraphQL

Database
- MongoDB, via MLabs

Testing
- GraphiQL

Sample query:

{
    book{
        name
        genre
        author{
            name
            age
        }
    }
}

returns an object "data" that holds an object, books, which holds an array of data
